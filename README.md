# BDIX CloudStream Extensions

Repository for DhakaFlix and FtpBd providers.

## Installation

1. Open CloudStream 3.
2. Go to **Settings** -> **Extensions**.
3. Click on **Add Repository**.
4. Enter the following URL:
   `https://raw.githubusercontent.com/salmanbappi/cloudstream-extensions/main/repo.json`
5. Install the **DhakaFlix**, **FtpBd**, or **Dflix** extension.

## Shortcut

If you have the **CloudStream Shortcut** (CloudStream Protocol) enabled, you can use:
`cloudstreamrepo://raw.githubusercontent.com/salmanbappi/cloudstream-extensions/main/repo.json`
